# Dan Kennedy Swipe File

Classic headline formulas, offer structures, and proven copy patterns.

## Headline Formulas

### Benefit-Driven Headlines

```
How to [Achieve Desired Result] Without [Pain Point]
```
Example: "How to Generate 50 Leads Per Week Without Cold Calling"

```
The [Number] [Timeframe] [System/Method] for [Desired Result]
```
Example: "The 7-Day System for Doubling Your Email Open Rates"

```
[Number] Ways to [Achieve Result] (Even If [Objection])
```
Example: "5 Ways to Close High-Ticket Clients (Even If You Hate Selling)"

### Curiosity Headlines

```
The [Adjective] Secret [Audience] Use to [Achieve Result]
```
Example: "The Counterintuitive Secret Top Realtors Use to Get Listings"

```
Why [Common Belief] Is Dead Wrong (And What to Do Instead)
```
Example: "Why 'Post Every Day' Is Dead Wrong (And What to Do Instead)"

```
What [Experts/Gurus] Don't Want You to Know About [Topic]
```
Example: "What Marketing Agencies Don't Want You to Know About Facebook Ads"

### Problem-Focused Headlines

```
Are You Making These [Number] [Topic] Mistakes?
```
Example: "Are You Making These 3 Deadly Landing Page Mistakes?"

```
Warning: [Problem] Could Be [Consequence]
```
Example: "Warning: Your Website Could Be Costing You $10K/Month in Lost Sales"

```
[Pain Point]? Here's the Fix That Works in [Timeframe]
```
Example: "No Leads? Here's the Fix That Works in 14 Days"

### Specificity Headlines

```
How [Specific Person] [Achieved Specific Result] in [Specific Timeframe]
```
Example: "How a Struggling Coach Generated $127,000 in 90 Days"

```
[Exact Number] [People] Can't Be Wrong: [Benefit]
```
Example: "4,327 Business Owners Can't Be Wrong: This System Works"

```
Get [Specific Result] or [Guarantee]
```
Example: "Get 25 Qualified Appointments in 30 Days or Your Money Back"

## Offer Templates

### The Irresistible Offer Formula

**Core Offer:**
"When you [take action], you get [main deliverable] that helps you [primary benefit]."

**Value Stack:**
- Main product/service (Value: $X)
- Bonus #1: [Complementary item] (Value: $X)
- Bonus #2: [Accelerator item] (Value: $X)
- Bonus #3: [Support item] (Value: $X)
- Total Value: $X,XXX

**The Ask:**
"You won't pay $X,XXX. You won't even pay $XXX. Today, you get everything for just $XX."

**Risk Reversal:**
"And if you don't [achieve result] within [timeframe], you get a full refund. No questions asked."

### The "Reason Why" Offer

Always give a reason for the deal:
- "Because I'm launching this program..."
- "Because I want 50 case studies..."
- "Because my calendar has 3 openings this month..."
- "Because I'm testing a new approach..."

A reason—even a weak one—increases response dramatically.

### The Deadline Stack

Layer multiple urgency elements:
1. **Hard deadline**: "Doors close Friday at midnight"
2. **Bonus deadline**: "First 20 get the advanced module free"
3. **Price deadline**: "Price goes up by $200 on Monday"
4. **Calendar deadline**: "We start the cohort on the 15th"

## Copy Patterns

### The "If/Then" Pattern

"If you're [situation], then you already know [problem]. And you've probably tried [failed solutions]. But what if there was a way to [desired result] without [pain point]?"

### The "Imagine" Pattern

"Imagine waking up and checking your phone to see [positive scenario]. Imagine [another positive outcome]. Now imagine [the ultimate result]. This isn't a fantasy. It's exactly what happens when you [use the solution]."

### The "Problem Amplification" Pattern

"Every day you don't solve [problem], you're losing [specific cost]. That's [monthly cost] per month. [Yearly cost] per year. And it only gets worse because [compounding factor]."

### The "Social Proof" Pattern

"When [credible person] first tried [solution], they were skeptical. Just like you might be right now. But within [timeframe], they [specific result]. Then [another person] got [different result]. And [third person] finally [breakthrough moment]."

### The "Bridge" Pattern

**Before:** [Current painful state with vivid details]
**After:** [Desired state with specific outcomes]
**Bridge:** [Your solution that gets them from before to after]

## Testimonial Formulas

### The Before/After/Bridge

"Before [solution], I was [struggling with problem]. I tried [other solutions] but nothing worked. Then I [discovered/used solution] and within [timeframe], I [specific result]. Now I [ongoing benefit]."

### The Skeptic Convert

"I was skeptical at first. I'd tried [other solutions] before. But [solution] was different because [reason]. Within [timeframe], I [result]. I only wish I'd [started sooner/found it earlier]."

### The Specific Number

"In just [timeframe], I [achieved specific number result]. That's [comparison to previous state]. And it only took [effort level]."

### The Credential Builder

"As a [relevant credential], I've seen a lot of [category]. [Solution] is the first one that actually [specific benefit]. I recommend it to all my [audience type]."

## Copy Length Rules

| Price Point | Copy Length | Format |
|-------------|-------------|--------|
| Free | Short (300-500 words) | Bullet points, quick benefits |
| $1-$50 | Medium (500-1000 words) | Benefits + light proof |
| $50-$500 | Long (1000-3000 words) | Full PAS + testimonials |
| $500-$2000 | Very Long (3000-5000 words) | Complete sales letter |
| $2000+ | Ultra Long (5000+ words) | Multi-page sales letter + video |

The more trust required, the more copy needed. Every sentence must earn its place.
